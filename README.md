magisk-module-smallstatusbar

Change the status bar width to 30px or 40px.
Confirmed to work on Android 16 AOSP.
You can also change 30.0px or 40.0px values to adjust the width to your preference.
This is fork of the magisk-module-nostatusbar.
https://github.com/AndroPlus-org/magisk-module-nostatusbar
